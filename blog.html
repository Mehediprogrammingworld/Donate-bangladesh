What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for representing HTML, XML, and other structured documents as a tree of nodes. Each node represents an element, attribute, text, or comment within the document. What is event delegation in the DOM, and why is it useful?

Event Delegation is a technique in JavaScript where a single event listener is attached to a parent element, and the event is propagated up the DOM tree to its children. This allows you to handle events for multiple child elements without having to attach individual event listeners to each one.
